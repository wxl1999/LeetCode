# 213. 打家劫舍 II

[题目链接](https://leetcode.cn/problems/house-robber-ii/description/)

## 解法一：动态规划

### 思路

- 考虑偷第一间，那么范围是 [0, n - 2]
- 考虑不偷第一间，那么范围是 [1, n - 1]
- 然后打家劫舍

### 代码

```py
class Solution:
    def rob(self, nums: List[int]) -> int:
        ans = nums[0]

        a = b = 0
        for num in nums[:-1]:
            c = max(b, a + num)
            a = b
            b = c
        ans = max(ans, b)

        a = b = 0
        for num in nums[1:]:
            c = max(b, a + num)
            a = b
            b = c
        ans = max(ans, b)

        return ans
```

### 复杂度

- 时间：O(n)
- 空间：O(1)
