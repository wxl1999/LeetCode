# 50. Pow(x, n)

[题目链接](https://leetcode.cn/problems/powx-n/description)

## 解法一：快速幂

### 思路

- x^n = n 个 x 相乘 = n 的二进制表示每一位 i * x^(2^i)

### 代码

```py
class Solution:
    def myPow(self, x: float, n: int) -> float:
        def mul(a, b):
            ans = 0

            t = 1
            while b:
                if b & t == 1:
                    ans += a
                a += a
                b >>= 1

            return ans

        ans = 1
        if n < 0:
            n = -n
            x = 1 / x

        while n:
            if n & 1:
                ans = mul(ans, x)
            x = mul(x, x)
            n >>= 1
        
        return ans
```

### 复杂度

- 时间：O(logn)
- 空间：O(1)
