# 287. 寻找重复数

[题目链接](https://leetcode.cn/problems/find-the-duplicate-number/description/)

## 解法一：双指针

### 思路

- 如果我们把数组变成一个图，如果这就变成了找到有环图的入口
  - 构建 nums[i] -> nums[nums[i]] 的连边

### 代码

```py
class Solution:
    def findDuplicate(self, nums: List[int]) -> int:
        s = nums[0]
        f = nums[nums[0]]
        while s != f:
            s = nums[s]
            f = nums[nums[f]]
        
        s = 0
        while s != f:
            s = nums[s]
            f = nums[f]
        return s
```

### 复杂度

- 时间：O(n)
- 空间：O(1)
