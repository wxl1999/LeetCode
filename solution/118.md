# 118. 杨辉三角

[题目链接](https://leetcode.cn/problems/pascals-triangle/description)

## 解法一：动态规划

### 思路

f[i][j] = f[i - 1][j - 1] + f[i - 1][j]

### 代码

```py
class Solution:
    def generate(self, numRows: int) -> List[List[int]]:
        f = [[1 for _ in range(i + 1)] for i in range(numRows)]
        for i in range(2, numRows):
            for j in range(1, i):
                f[i][j] = f[i - 1][j - 1] + f[i - 1][j]

        return f
```

### 复杂度

- 时间：O(n^2)
- 空间：O(n^2)
