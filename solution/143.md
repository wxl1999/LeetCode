# 143. 重排链表

[题目链接](https://leetcode.cn/problems/reorder-list/description/)

## 解法一：双指针

### 思路

- 先找到中间节点
- 然后翻转右半段
- 最后将两个链表逐个接上

### 代码

```py
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def reorderList(self, head: Optional[ListNode]) -> None:
        """
        Do not return anything, modify head in-place instead.
        """
        s = head
        f = head.next
        while f and f.next:
            s = s.next
            f = f.next.next

        p = None
        q = s.next
        s.next = None
        while q:
            t = q.next
            q.next = p
            p = q
            q = t   

        h1 = head
        h2 = p
        while h1 and h2:
            t1 = h1.next
            t2 = h2.next

            h1.next = h2
            h2.next = t1

            h1 = t1
            h2 = t2
```

### 复杂度

- 时间：O(n)
- 空间：O(1)
