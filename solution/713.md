# 713. 乘积小于 K 的子数组

[题目链接](https://leetcode.cn/problems/subarray-product-less-than-k/description/)

## 解法一：双指针

### 思路

- 连续子数组可考虑滑动窗口
- 如何一个子数组乘积小于 k，则不断右移左指针，子数组乘积仍然小于 k

### 代码

```py
class Solution:
    def numSubarrayProductLessThanK(self, nums: List[int], k: int) -> int:
        ans = 0

        n = len(nums)

        l = 0
        r = 0
        p = 1
        while r < n:
            p *= nums[r]
            while l <= r and p >= k:
                p //= nums[l]
                l += 1
            ans += (r - l + 1)
            r += 1

        return ans
```

### 复杂度

- 时间：O(n)
- 空间：O(1)
