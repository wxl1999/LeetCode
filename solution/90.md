# 90. 子集 II

[题目链接](https://leetcode.cn/problems/subsets-ii/description/)

## 解法一：DFS

### 思路

- 去重考虑先排序
- 如果不选某个元素，就都不选

### 代码

```py
class Solution:
    def subsetsWithDup(self, nums: List[int]) -> List[List[int]]:
        nums.sort()
        ans = []
        path = []
        n = len(nums)

        def dfs(i):
            if i == n:
                ans.append(path[:])
                return

            x = nums[i]

            path.append(x)
            dfs(i + 1)
            path.pop()

            i += 1
            while i < n and nums[i] == x:
                i += 1
            dfs(i)

        dfs(0)
        return list(ans)
```
