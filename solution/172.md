# 172. 阶乘后的零

[题目链接](https://leetcode.cn/problems/factorial-trailing-zeroes/description)

## 解法一：数学

### 思路

- 求质因子2和5个数的最小值
  - 5的个数 <= 2，所以就是求 5 的个数

#### 代码一

```py
class Solution:
    def trailingZeroes(self, n: int) -> int:
        ans = 0
        for i in range(5, n + 1, 5):
            while i % 5 == 0:
                i //= 5
                ans += 1
        return ans
```

#### 代码二

每次统计有多少个数包含 k 个质因子 5

```py
class Solution:
    def trailingZeroes(self, n: int) -> int:
        ans = 0
        while n:
            n //= 5
            ans += n
        return ans
```

### 复杂度

- 时间：O(n)
- 空间：O(1)
