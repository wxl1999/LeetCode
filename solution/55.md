# 55. 跳跃游戏

[题目链接](https://leetcode.cn/problems/jump-game/description/)

## 解法一：贪心

### 思路

- 维护当前可以到达的最远距离

### 代码

```py
class Solution:
    def canJump(self, nums: List[int]) -> bool:
        n = len(nums)
        r = 0
        for i in range(n):
            if i <= r:
                r = max(r, i + nums[i])
                if r >= n - 1:
                    return True
            else:
                return False

        return False
```

### 复杂度

- 时间：O(n)
- 空间：O(1)
