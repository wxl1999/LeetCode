# 53. 最大子数组和

[题目链接](https://leetcode.cn/problems/maximum-subarray/description/)

## 解法一：动态规划

### 思路

- 记 f(i) 为以 i 为结尾的所有子数组的最大和
  - 有 f(i) = max(f(i - 1) + nums[i], nums[i])
  - 因为只和 f(i - 1) 有关，使用滚动数组优化空间

### 代码

```py
class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        pre = 0
        ans = nums[0]

        for num in nums:
            pre = max(pre + num, num)
            ans = max(ans, pre)

        return ans
```

### 复杂度

- 时间：O(n)
- 空间：O(1)
