# 69. x 的平方根 

[题目链接](https://leetcode.cn/problems/sqrtx/description/)

## 解法一：二分

### 思路

- 找到最大的正整数 k，使得 k * k <= x

#### 代码一

```py
class Solution:
    def mySqrt(self, x: int) -> int:
        l = 0
        r = x
        while l < r:
            mid = l + (r - l + 1) // 2
            if mid ** 2 <= x:
                l = mid
            else:
                r = mid - 1

        return l
```

#### 代码二（浮点数）

- eps 一般取比要求精度再多两个小数点
- 左右端点取数据范围的边界
  - 对于 [0, 1] [-1, 0]，其平方根比自身要大，边界需要取到 ±1

```py
eps = 1e-8

def func(n):
    l, r = min(n, -1), max(n, 1)
    while r - l > eps:
        mid = (l + r) / 2
        if mid ** 3 >= n:
            r = mid
        else:
            l = mid
    return l
```

### 复杂度

- 时间：O(logn)
- 空间：O(1)
