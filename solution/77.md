# 77. 组合

[题目链接](https://leetcode.cn/problems/combinations/description)

## 解法一：DFS

### 思路

- 枚举每个数选或不选

### 代码

```py
class Solution:
    def combine(self, n: int, k: int) -> List[List[int]]:
        ans = []
        path = []

        def dfs(i):
            if len(path) == k:
                ans.append(path[:])
                return
            elif n - i < k - len(path):
                return

            dfs(i + 1)

            path.append(i + 1)
            dfs(i + 1)
            path.pop()

        dfs(0)
        return ans
```

### 复杂度

- 时间：O(2^n)
- 空间：O(n)
