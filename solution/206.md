# 206. 反转链表

[题目链接](https://leetcode.cn/problems/reverse-linked-list/description/)

## 解法一：头插法

### 思路

- 每次把当前节点插入新链表的头部

### 代码

```py
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def reverseList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        h = ListNode()
        n = head
        while n:
            t = n.next
            n.next = h.next
            h.next = n
            n = t
        return h.next
```

### 复杂度

- 时间：O(n)
- 空间：O(1)
