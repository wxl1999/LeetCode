# 633. 平方数之和

[题目链接](https://leetcode.cn/problems/sum-of-square-numbers/description/)

## 解法一：双指针

### 思路

- 等价于给定有序数组 [0, 1, 4, ..., k^2]，判断是否存在两数之和等于 c

### 代码

```py
class Solution:
    def judgeSquareSum(self, c: int) -> bool:
        a = 0
        b = isqrt(c)
        while a <= b:
            s = a ** 2 + b ** 2
            if s == c:
                return True
            elif s < c:
                a += 1
            else:
                b -= 1
        return False
```

### 复杂度

- 时间：O(n)
- 空间：O(1)
