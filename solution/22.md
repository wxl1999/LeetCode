# 22. 括号生成

[题目链接](https://leetcode.cn/problems/generate-parentheses/description/)

## 解法一：递归

### 思路

- 牢记左括号数量需要≥右括号数量

### 代码

```py
class Solution:
    def generateParenthesis(self, n: int) -> List[str]:
        ans = []
        path = []

        def dfs(l, r):
            if l == n and r == n:
                ans.append(''.join(path))
                return

            if l < n:
                path.append('(')
                dfs(l + 1, r)
                path.pop()
            if l > r:
                path.append(')')
                dfs(l, r + 1)
                path.pop()

        dfs(0, 0)
        return ans
```

### 复杂度

- 时间：见题解
- 空间：O(n)
