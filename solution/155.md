# 155. 最小栈

[题目链接](https://leetcode.cn/problems/min-stack/description/)

## 解法一：模拟

### 思路

- 一个栈存当前值
- 一个栈存最小值

### 代码

```py
class MinStack:

    def __init__(self):
        self.s = []
        self.min_s = []

    def push(self, val: int) -> None:
        self.s.append(val)

        if len(self.min_s) > 0:
            val = min(val, self.min_s[-1])
        self.min_s.append(val)

    def pop(self) -> None:
        self.s.pop()
        self.min_s.pop()

    def top(self) -> int:
        return self.s[-1]

    def getMin(self) -> int:
        return self.min_s[-1]


# Your MinStack object will be instantiated and called as such:
# obj = MinStack()
# obj.push(val)
# obj.pop()
# param_3 = obj.top()
# param_4 = obj.getMin()
```

### 复杂度

- 时间：O(n)
- 空间：O(1)
